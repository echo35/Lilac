Files in the $data/shared can be loaded with /?p=file&path=, such as /?p=file&path=img/alley.jpg:



The /?p=file script automatically sets mime type, and its behavior can be changed in the $hidden/resources/sharing.py source file.


Backend bash-scripts such as $data/scripts/hello_world.sh can be executed using /?p=exec&path=hello_world.sh&args=echo35

Python scripts are also supported, like the string-to-sha256sum script at /?p=exec&path=calc_sha256.py&args=helloworld:
